# Temperature Converter 🌡️

A simple and clean web app to convert temperatures 
between Celsius, Fahrenheit, and Kelvin instantly.

## Features
- Convert between Celsius, Fahrenheit, and Kelvin
- Clean and responsive UI
- Instant result on button click

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## How to Use
1. Enter a temperature value
2. Select the unit to convert FROM
3. Select the unit to convert TO
4. Click Convert button
5. See the result instantly!

## Formulas Used
- Celsius to Fahrenheit → (C × 9/5) + 32
- Fahrenheit to Celsius → (F - 32) × 5/9
- Celsius to Kelvin → C + 273.15
- Kelvin to Celsius → K - 273.15
